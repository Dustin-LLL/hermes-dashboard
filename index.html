Token 配置
📋 获取 GitHub Token 步骤：
  1. 登录 GitHub 账号 → 点击右上角头像 → Settings
  2. 左侧菜单底部 → Developer settings
  3. 选择 Personal access tokens → Tokens (classic)
  4. 点击 Generate new token (classic)
  5. 设置名称、过期时间，勾选所需权限：
    • repo - 完整仓库访问（用于 PR、代码操作）
    • workflow - GitHub Actions
    • read:user - 读取用户信息
  6. 点击 Generate token → 复制生成的 Token
